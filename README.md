# Diary 

Вебдодаток для ведення персональних записів, побудований на архітектурі MVC.


## Функціонал

* **Авторизація:** реєстрація та вхід через сесії.
* **Нотатки:** повний цикл CRUD (створення, читання, редагування, видалення).
* **Безпека:** захист від SQL-ін’єкцій через підготовлені запити PDO.
* **Адміністрування:** окрема панель для керування списком користувачів.


## Архітектура

* `/app` — Контролери, Моделі та Представлення.
* `/core` — Ядро системи (Маршрутизатор, Singleton БД).
* `/public` — Точка входу, стилі та конфігурація сервера.
* `/config` — Налаштування підключення до бази даних.


## Технологічний стек

* **Backend:** PHP 8.x
* **База даних:** MySQL
* **Frontend:** Чистий CSS3 (модульна структура)
* **Сервер:** Apache (.htaccess)


## Встановлення

1. **База даних:** Імпортуйте SQL-дамп у вашу базу даних MySQL.
2. **Конфігурація:** Вкажіть параметри доступу (хост, назва БД, логін, пароль) у файлі `/config/config.php`.
3. **Сервер:** Налаштуйте ваш вебсервер (Apache/Nginx) на корінь папки `/public`.
4. **Доступ:** Переконайтеся, що модуль `mod_rewrite` увімкнено для коректної роботи маршрутизації через `.htaccess`.
